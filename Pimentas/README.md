# Pepper Atlas

Projeto acadêmico desenvolvido com HTML, CSS e JavaScript para apresentar um atlas visual de pimentas do Brasil e do mundo.

O site permite explorar variedades por ardência, origem, perfil de sabor e uso gastronômico, além de abrir detalhes de cada pimenta em uma janela modal.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Cards dinâmicos de pimentas
- Filtros por ardência, origem, sabor e uso
- Busca por nome, origem ou perfil de sabor
- Modal com detalhes de cada pimenta
- Seção especial para pimentas brasileiras
- Marcação de favoritas com `localStorage`
